연결 자산 검색

기존 Library를 검색하고 관계와 활용 내용을 기록합니다.

게시 전 필수

회사 내부 자산 링크

원본·근거·결과물의 위치와 접근 범위를 구조화해 연결합니다.

게시 전 필수
직접 링크 추가 추천 목록에 없는 원본·근거·결과물도 여러 개 추가할 수 있습니다.
연결된 링크 자동 추천과 직접 추가 링크를 함께 확인합니다.

등록 전 검증

검증 후 사내 Library에 등록하고 Reviewer 검토를 요청합니다.

등록 후 게시 순서

  1. 1
    Library 등록

    검증을 통과한 자산을 사내 Library에 등록합니다.

  2. 2
    Reviewer 검토

    등록과 동시에 Reviewer에게 검토를 요청합니다.

  3. 3
    승인·게시

    Reviewer 승인 후 Library 검색과 상세 화면에 게시됩니다.
