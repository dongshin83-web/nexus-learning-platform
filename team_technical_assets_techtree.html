Technical Growth Path

내가 무엇을 어떤 순서로 배워야 하는지 보이게 한다

팀 교육 과정이 따로 있더라도, Tech Tree는 개인이 기술적으로 무엇을 먼저 이해하고 어떤 실습과 산출물을 남겨야 하는지 확인하는 자기 점검용 경로입니다.

Role 공식 교육 과정의 대체가 아니다

교육 과정과 Library 사이를 이어주는 학습 순서 지도입니다.

Output 배움의 결과를 남긴다

각 단계는 읽어야 할 내용보다 남겨야 할 산출물을 함께 제시합니다.

기술 성장 단계

결과를 읽는 법에서 시작해 모델링, 기술영역별 메커니즘, CoR/VD Request, BP/Manual, AI/타 Domain 연계로 확장합니다.

Tech Tree와 Library의 연결

학습 단계마다 Library에서 찾아야 할 자료 유형이 다릅니다.

초기 학습

교육자료와 Tool Manual을 먼저 보고, 기존 모델을 재현합니다.

기술영역 이해

BP와 CoR 보고서를 통해 문제 메커니즘과 판단 기준을 읽습니다.

과제화

CoR 제안서와 VD Request 결과를 참고해 요청을 과제로 전환합니다.

자산화

새로운 경험을 BP, Tool Manual, 교육 활용 메모로 다시 Library에 남깁니다.

Tech Tree is a learning path, not a formal curriculum replacement.
